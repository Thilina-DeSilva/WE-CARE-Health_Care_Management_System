# 🏥 WE CARE - Hospital Management System (HMS)

A modern Hospital Management System designed to handle all major hospital operations including administration, patient handling, pharmacy POS, and laboratory management.

> 🚧 This project is currently under development.

---

## 🚀 Features

### 👨‍💼 Admin Panel
- User Management (Auto ID + Password generation)
- Doctor Management with specialization filtering
- Stock Monitoring (Medicines, Services, Lab)
- Reports & Analytics (Revenue, Sales)
- Full CRUD operations

---

### 🧑‍💻 Receptionist Panel
- Patient Registration with:
  - Auto-generated Patient ID
  - Printable ID Card (with photo)
- Appointment Scheduling:
  - Select patients from dropdown
  - Filter doctors by specialization
  - Only available time slots allowed
- Consultation fee handling

---

### 💊 Pharmacist POS System
- Fast billing system
- Auto-fetch patient & consultation data
- Add medicines and services
- Real-time total calculation
- Generate final receipt

---

### 🧪 Laboratory System
- Lab test request management
- Generate lab reports
- Send lab charges to POS
- Printable reports (In progress)

---
